
Purpose

The purpose of this capstone project is to design, develop, and evaluate a web-based online ordering system for brand-new and preloved clothing that streamlines product listing, checkout, and buyer–seller management. The system aims to:

- Provide an accessible, user-friendly marketplace where sellers can list new and preloved garments with clear descriptions, images, categories, and condition indicators.
- Implement a secure and efficient product browsing and search experience that supports filters (size, brand, price, condition), sorting, and detailed product pages to help buyers make informed choices.
- Offer a reliable checkout flow with cart management, multiple payment options, order confirmation, and basic fraud-prevention measures to ensure secure transactions.
- Enable effective buyer–seller management features, including user profiles, messaging or inquiry tools, order tracking, seller dashboards (inventory and sales overview), and basic dispute resolution support.
- Reduce barriers for casual sellers of preloved items by simplifying listing creation and inventory updates while giving established sellers tools to manage catalogs and orders.
- Improve sustainability and circular fashion practices by promoting the resale of quality preloved clothing, extending garment life, and reducing textile waste.
- Evaluate system usability, performance, and user satisfaction through testing with prospective buyers and sellers to identify improvements and validate the platform’s effectiveness.

This project will result in a functional prototype and supporting documentation (requirements, design, implementation details, and evaluation results) demonstrating how an integrated online ordering platform can meet user needs for buying and selling both brand-new and preloved apparel.
